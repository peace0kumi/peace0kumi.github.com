마크업 폴더 구조 참고사항

/css

분류 설명 예시
/front 페이지 관련 css를 포함하며, 'front.기능.css'로 네이밍 한다. front.temp.css
/layout 레이아웃 관련 css를 포함하며, 'layout.기능.css'로 네이밍 한다. layout.front.css
/module 모듈 관련 css를 포함하며, 'module.기능.css'로 네이밍 한다. module.button.css

/html

분류 설명 예시
/front 실제 작업 페이지를 포함하며, 문서 네이밍은 'ScreenID.html'로 작성한다. MAIN.01.00.00.00.html
/template 페이지의 전반적인 레이아웃 문서들을 포함한다.

/images

분류 설명 예시
/@database 프로모션 배너, 날짜가 포함된 이벤트성 이미지 등 임시로 사용하는 이미지를 포함한다. temp_promotion_img.png, temp_banner_img.png
/@thumbnail 관리자 페이지에서 등록하는 이미지를 포함한다.
/front 해당 카테고리에 필요한 이미지를 카테고리 네임으로 폴더를 구분하여 관리한다.
/global 라디오 버튼, 체크박스, 탭 등 모듈에 사용하는 이미지를 포함한다.
/layout 레이아웃에 사용하는 이미지를 포함한다.
/main 로고, BI, CI 등 프로젝트에 메인으로 사용하는 컨텐츠 이미지들을 포함한다.
