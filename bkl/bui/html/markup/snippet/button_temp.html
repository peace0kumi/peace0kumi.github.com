목차

snippet

button

basic system text button

module prefix "btn"으로 element의 기본 속성이 reset 됩니다.

사용목적에따라 화면 이동(anchor), 서식 전송(submit), 동작 제어(button)등 역할(element)에 맞게 마크업 하며, <input type="submit" />은 표현의 한계로 <button type="submit">으로 대체합니다.

아웃라인(outline) 대비 크기를 결정 후 상태(state)값에 따라 컬러를 적용합니다.

bui-toggle과 함께 사용할 수 있습니다.

버튼데모

markup

anchor <a href="#" class="btn">
submit <button type="submit" class="btn">
button <button type="button" class="btn">

option

module btn
theme theme-a
type type-a
outline x-small, small, medium, large, x-large
level fall-01 to fall-03, rise-01 to rise-04
symbols before-*, after-*
function bui-toggle
