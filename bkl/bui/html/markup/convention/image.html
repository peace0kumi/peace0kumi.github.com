아이콘 이미지 제작

아이콘 이미지는 기본 48*48 사이즈의 png 파일로 제작한다.

네이밍

네이밍 조합

  1. 이미지 네이밍은 이미지 확장자와 무관하게 적용한다.
  2. 임시 이미지는 'temp_' 를 조합한다.
  3. 이미지 이름은 '형태_의미_상태' 순서로 조합한다
  4. '_' 언더바 는 2개를 초과하여 조합하지 않는다
  5. 같은 분류의 이미지가 두개 이상이면 파일 이름 마지막에 숫자를 추가하여 구분한다.
  6. 임의로 축약하지 않는다
  7. 영문 소문자만을 사용하며, 숫자로 시작하지 않는다.
