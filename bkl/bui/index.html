법무법인(유한)태평양 모바일 웹 사이트 구축을 위한 BUI(Basic User Interface)입니다.

BUI(Basic User Interface)의 목적은 마크업 개발자가 지켜야 할 기준을 제시함으로써,
협력 부서 간의 이해도를 높여 프로젝트의 원활한 진행을 돕고 더욱 높은 수준의 결과물을 내는 것입니다.

말하자면 BUI(Basic User Interface)에서 정의한 규칙에 따라 생산되는 결과물은,
일관성 있고 최적화된 코드로 불필요한 자원을 줄이게 됩니다.

또한, 향후 유지관리 시 작업능률 향상과 생산성 제고를 극대화할 것입니다.
